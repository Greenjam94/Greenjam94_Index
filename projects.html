Cowrie Honeypot

My first honeypot experience. Cowrie is a SSH honeypot based on Kippo. I have done none of the dev work for this. What makes this a project for me is that this is my first one with a security focus. Cowrie is configured to set up and log any SSH traffic, our instantiation of Cowrie has already caught a few files that a rouge user has tried to upload and execute. These files are then used for review to see what the person (or bot) was trying to do to our server.
